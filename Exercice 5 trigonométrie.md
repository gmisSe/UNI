# 1
Dans le triangle rectangle en H.
$\cos_{60}=\frac{BH}{16}$
$BH = 16 \cos_{60}= 8$
$BH = 8km$

# 2
Dans le meme triangle 
$\sin_{60}= \frac{AH}{16}$
$AH =16 \sin_{60}=13.856km$

# 3
Dans BHA :
BAH + 60 + 90 = 180
BAH = 180 - 150 = 30
BAH + HAM = BAM
30 + HAM = 75
HAM = 75 - 30 = 45 

Dans HAM rectangle en H
AMH + 90 + 45 = 180
AMH = 180 - 135 = 45
HAM est donc isocèle 
ses deux cotés sont égaux 
HM = AH = 13.856km
